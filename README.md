## Instalation
1. Install the theme using your favorite vim plug in manager or manually place in `colors/neotrix.vim` directory in  `~/.vim/colors`
2. Add the following to your `init.vim` or `.vimrc`:
  ```vim
  syntax on
  colorscheme neotrix
  set termguicolors
  ```
